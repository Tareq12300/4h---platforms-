# 📡 كريبتو سكانر — بوت تليغرام

بوت يراقب **1000 عملة** من CoinMarketCap مع تحليل RSI وMACD وحجم التداول.

## الأوامر

| الأمر | الوصف |
|-------|-------|
| `/scan` | سكانر الفرص (RSI + MACD + حجم) |

## طريقة الرفع على Railway

### ١. احصل على المفاتيح

**مفتاح CMC:**
1. اذهب إلى https://coinmarketcap.com/api
2. سجّل حساب مجاني
3. انسخ الـ API Key

**توكن البوت:**
1. افتح تليغرام → ابحث عن @BotFather
2. أرسل `/newbot`
3. اختر اسم وusername للبوت
4. انسخ الـ Token

### ٢. ارفع على Railway

1. اذهب إلى https://railway.app وسجّل دخول بـ GitHub
2. اضغط **New Project → Deploy from GitHub repo**
3. ارفع الملفات لـ GitHub repo جديد
4. في Railway اضغط على المشروع → **Variables** وأضف:
   ```
   CMC_API_KEY = مفتاحك هنا
   TELEGRAM_BOT_TOKEN = توكن البوت هنا
   ```
5. اضغط **Deploy** — البوت يشتغل خلال دقيقة!

### ٣. تأكد إنه شغّال

افتح البوت في تليغرام وأرسل `/start`
